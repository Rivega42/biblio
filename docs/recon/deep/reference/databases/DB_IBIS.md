## 3. Рабочие листы ввода (`.ws`)

### 3.1 Механизм выбора (`WS.OPT`/`WS31.OPT`, по полю 920, метод 5)
`WS.OPT`: `PAZK→PAZK`, `PVK→PVK`, `SPEC→SPEC`, `J→!RPJ`, `NJ/NJP/NJK→!NJ`, `AUNTD→AUNTD`, `ASP→ASP`, `MUSP→MUSP`, по умолчанию `+++++→IBIS`. `WS31.OPT` — те же ключи → листы с суффиксом 42/51/31.

### 3.2 Формат `.ws` (по `PAZK.WS`, `IBIS.WS`)
Шапка: `N_страниц`, N имён страниц, `N_полей`. Поле — блок 9 строк: `[1]метка | [2]подпись | [3]повтор(0/1/2) | [4]код-редактора (или 8-знач. контекст 8MMMTTT) | [5]доп.параметр(.wss / .mnu\\режим / ,префикс=,формат) | [6]ФЛК(@формат) | [7]подсказка | [8]— | [9]значение по умолчанию`.
Коды редактора: `0`текст, `1`меню(.mnu), `2`словарь(,префикс=,), `3`справочник с маской, `4`многострочный текст, `5`расширенный ввод по .wss, `6`иерархич.справочник(.tre), `7`да/нет, `10`ввод через Authority-файл, `11`тезаурус, `16`выбор файла.

### 5.2 Меню форматов `PFTW.MNU`
KN/MN/ASP/MUSP (БО книг/томов/аналитики/музея), комплект КК (!KKKOW основная, !KKKI/!KKKIMH оборот, !KKKDW добавочные, !KKKC ссылочные, !KKKUW/!KKKT для систематич./топографич., !KKKFW формуляры, !KKKFH штрих-коды), JW/NJW/NJTPW (журнал/номер/с техпутём), BRIEF, UMARCW/USMRCW/RMARCW (MARC), INFOW (информационный), PERSW (персоналии), BOKO_H (БО+книгообеспеч.), ALL, Bar_Kod_H (штрих-коды), DUBLINC_ (Dublin Core), IBISKW (для Комплектатора), KMA_Full, kk_pth_* (КК HTML), optim_h_iri (для эл.почты).

### 5.3 Разбор ключевых форматов
**`BRIEF.PFT`** (ветвление по 920): `920='SZPRF'`→служебная запись проверки фонда; иначе свёрнутое одностроч.БО (503→461→автор+200→выходные→эл.ресурс→объём); для `920:'NJ'` через `ref(l('I='v933))` подтягивает заглавие журнала+год/том/№; для аналитики — «// источник» из 463/963.
**`KN.PFT`** (`…Deposit\KN.PFT`, главный полный для книг) — модульный диспетчер (RTF). Если `907^c:'ОБРНЗ'`→плашка «Издание в обработке». Вызывает подпрограммы (дерево `Pft.tre`): `6osvex`(свободные экз.), `6orchifr`(расст.шифр), `6ozboz`(заголовок БО), `Otitkn`(заглавие), `oizd`(издание), `ovixd`(выходные), `ofizxar`(физ.хар.), `oser`(серия), `oprim`(примечания), `oISBNcn`(ISBN+цена), `ogrudk`(УДК/ГРНТИ), `orubdks`(рубрики), `oprip481`(подборка), `oex910`(экземпляры), `ofinal`, `bibl390`(цитирование).
**`MN.PFT`** — как KN + `6otit461`(заглавие общей части МН) + `6ibis925`(тома).
**`ASP.PFT`** — аналитика: `6otitkn` + `oistasp`(источник статьи).
HTML-варианты (`_H`) — то же содержание с HTML-разметкой.

### 5.4 Прочие PFT (по назначению)
Экранно-индексные (формируют термины): `!910/!910W`(экз), `!920`(контроль 920), `!900/!901/!903/!907`(коды), `!600/!601/!606/!607`(рубрики), `!461/!463/!700D/!701/!702/!710D`(БО), `!330n/!454n/!922n/!925n/!481n/!488n`(паралл.заглавия). КК/формуляры: `!KKK*`, `!KD/!KDS/!KDSP/!KDUD`. Модификаторы глоб.корректировки: `!DMOD*`. Фасеты: `!F8AVT/!F8COL/!F8TIT/!F8ORG/!F8P/!F8SUB`. Счётчики: `!SCNT1…!SCNT10`.
