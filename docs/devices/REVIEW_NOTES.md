# REVIEW_NOTES — аудит документации docs/devices против декомпиляции

> Детальное ревью всех документов папки **против декомпилированного исходника** (`_scratch_devices/dc_*`) + дампов строк + конфигов стенда. Две независимые проверки (две группы документов) + перепроверка спорных правок. Дата: 2026-06-26.

## Вердикт

Фактическое **ядро точно**: полевая модель ИРБИС (поля/подполя/статусы), кадры SIP2 (номера/коды/CRC/sequence), контракт API (151 операция, 106 DTO), функции LAS/TagService, enum'ы, P/Invoke, протокол сокета :6001 — **сверены построчно, без crit/major фактических ошибок**. Реальные правки — **устаревшие места** (декомпиляция закрыла часть OPEN_QUESTIONS), несколько off-by-one и **уточнения проектных решений** в ТЗ/дизайне.

- **EASYBOOKABIS_IRBIS_MAP** — поля 28/30/24/103/40(подполя F/A/B/K/V/D/1/E/G/H/I/C/2/R/L)/910^A/903/999/12814/12815/100/101/51/31/3054, enum `BookState` значения, префиксы `RI/GUID/MDPASP/H/IN`, `arm=B/C`, все номера строк-цитат — **точны**.
- **EASYBOOKABIS_SIP2_MAP** — все кадры (`9300CN…`, `9901002.00AY`, `63015…`, `17…`, `11YN…|BIN|AY`, `09N…`, `15+…`, `29YN…`), коды полей, CRC `(0x10000-Σ)&0xFFFF`, sequence, маски summary (AU/AS/CD), overdue `Substring(41,4)` — **дословно совпадают**.
