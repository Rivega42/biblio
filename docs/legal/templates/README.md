## Как пользоваться
- Везде, где нужны реальные данные, стоят плейсхолдеры вида `<ЗАПОЛНИТЬ: …>`. Заполнить реальными реквизитами правообладателя/авторов.
- ❓ помечены значения, которые нужно сверить с первоисточником на дату подачи.
- **Не коммитить** в репозиторий заполненные версии с персональными/корпоративными данными — храните их вне репозитория.
