## 4. Статус

**Сделано (из архива):** полный каталог протокола; реестр 51 базы; механика конфигов (префиксы,
словарь, формы, форматы, связи, пути); GUAR-488; структура всех 6 настольных клиентов (источник функций); веб-модель вендора; модель
доступа; привязка функций к протоколу; инженерное ТЗ + аддендум доступа/полного набора функций.

**Полнота:** сильная карта на уровне структуры/механики; **не** исчерпывающая до каждого поля
(PFT/WSS/CHM сэмплированы). Достаточно для архитектуры и каркаса, не для прямой генерации боевого
кода по каждому полю.

**Осталось (вне архива — требует действий):**
1. **Проход Б — живой сервер `:6666`:** форматы пакетов/коды результата, права `RIGHT`, протокол очереди заказов. Запускать локально (WSL/Windows).
2. **Конфиги боевых баз СПб ГТБ** (PLAY/TUAR/EK/ESKIZ/HPO) — с сервера библиотеки; механику применить ту же.
3. **Независимая перепроверка** Клод Кодом по `PROMPT_CLAUDECODE_RECON.md` → сверка с этой картой.
4. **Уточнения под СПб ГТБ:** уровни/пресеты-роли; точные правила полей Каталогизатора (irbiscat.doc разд.4).
