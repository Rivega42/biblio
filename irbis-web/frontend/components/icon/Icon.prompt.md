Линейная иконка из локального набора ИРБИС-Веб — единственный способ ставить иконки в интерфейсе (не вставляйте сырые SVG и не используйте emoji).

```jsx
<Icon name="search" />
<Icon name="book" size={24} />
<Icon name="alert-triangle" size={18} label="Ошибка" />
```

- `name` — имя из набора (`ICON_NAMES` перечисляет все); 24×24, обводка `currentColor`, наследует цвет текста.
- `size` (по умолчанию 20), `strokeWidth` (1.75).
- Без `label` иконка декоративная (`aria-hidden`); с `label` — озвучивается скринридером.
- Статусы экземпляров всегда сопровождаются иконкой + текстом, не только цветом (`check-circle` / `clock` / `x-circle`).
