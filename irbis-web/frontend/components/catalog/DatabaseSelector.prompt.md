Иерархический мультиселектор баз (§1.1) — одно окно «Электронный каталог и Базы данных»: раскрываемые группы (ЭК, Базы Либретто), чекбоксы, «Выбрать все» / «Снять всё», без галочек по умолчанию. `value` — массив id, питает мультибазовый поиск.

```jsx
<DatabaseSelector
  databases={[
    {id:"EK", group:"ek", name:"Электронный каталог", icon:"book", count:214530},
    {id:"PERIO", group:"ek", name:"Периодические издания", icon:"newspaper", count:18740},
    {id:"SKETCH", name:"Эскизный фонд", icon:"images", count:8412},
    {id:"IMGOPERA", group:"libretto", name:"Либретто опер", icon:"music", count:2140},
  ]}
  groups={{ ek:{label:"Электронный каталог", icon:"book"}, libretto:{label:"Базы Либретто", icon:"music"} }}
  value={dbIds} onChange={setDbIds}
/>
```

- `value` пустой по умолчанию (нет галочек). Поиск требует ≥1 базы.
- Группы раскрываются; у группы тристейт-чекбокс «выбрать все вложенные».
- Кнопка показывает «Базы не выбраны» / название одной базы / «Выбрано баз: N».
