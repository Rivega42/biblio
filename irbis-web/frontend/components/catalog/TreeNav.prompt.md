Навигатор-классификатор (§4, §10) — дерево рубрик ГРНТИ/УДК/ББК с количеством записей; выбор узла фильтрует выдачу. Несколько классификаторов — вкладками.

```jsx
<TreeNav
  navigators={[
    { id:"grnti", label:"ГРНТИ", tree:[
      { code:"18", label:"Искусство", count:412, children:[
        { code:"18.45", label:"Театр", count:230 }]}]},
    { id:"udk", label:"УДК", tree:[ /* … */ ] },
  ]}
  value={navCode} onPick={(node)=> setNavCode(node ? node.code : null)}
/>
```

- Узлы со счётчиком — ленивая подача (count рядом).
- Повторный клик по выбранному узлу снимает выбор (`onPick(null)`).
