Контейнер под серверно-отрендеренный контент (PFT) с базовой типографикой. Санитайзинг — ДО вставки.

```jsx
<PftBlock html={record.pftHtml} sanitize={DOMPurify.sanitize} />
// или готовый React-контент:
<PftBlock label="Аннотация"><p>…</p></PftBlock>
```

⚠️ Никогда не вставляйте серверный HTML без очистки. В моках html уже обезличен и безопасен.
