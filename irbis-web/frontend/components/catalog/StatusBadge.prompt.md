Бейдж статуса экземпляра — цвет + иконка + текст (никогда не только цвет).

```jsx
<StatusBadge status="available" />   {/* Доступен */}
<StatusBadge status="issued" />      {/* Выдан */}
<StatusBadge status="unknown" size="sm" />
<StatusBadge status="available" dot /> {/* компактно: точка + текст */}
```
