# Сводный словарь полей/подполей формата ИРБИС64

> Источники: `IRBISPRL.DOC (txt:N)` — Таблица 1 (ЭК), Таблица 2 (комплектование); реальные рабочие листы `cfg_txt/IBIS/*.wss`, `*.ws`. Знак `+` = обязательность; `(mf)` = соответствие представлению UNIMARC; «повтор.» Y/N; «словарь» = префикс инверсии в поисковом словаре.

## Условные обозначения

| Графа | Содержание | Ссылка |
|---|---|---|
| Метка | Числовая метка поля | IRBISPRL.DOC (txt:151–152) |
| `^x` | Разделитель подполя (буква/цифра) | IRBISPRL.DOC (txt:151) |
| `+` | Обязательный элемент | IRBISPRL.DOC (txt:152) |
| повтор. | Поле повторяется (Y) | IRBISPRL.DOC (txt:153) |
| (mf) | Элемент представлен в UNIMARC | IRBISPRL.DOC (txt:154) |
| Словарь | Префикс термина в поисковом словаре (A= авторы, T= заглавия, M= коллективы, K= ключевые слова и др.) | IRBISPRL.DOC (txt:155–160, 166–176) |

Структура реального рабочего листа (`cfg IBIS/IBIS.WS`, `*.wss`): на поле — метка, имя, флаг повторяемости, флаг обязательности, код типа ввода, имя ресурса-помощника (.mnu/.wss/.tre); подполя описываются в одноимённом `.wss` (напр. поле 200 → `200si.wss`).
